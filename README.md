# Neural-Style-Transfer


## Overview

This is the code for my implementation of the neural-style transfer paper by Leon Gatys etal.


# Dependencies (pip install)

```
numpy
pandas
matplotlib
pytorch
PIL
scipy
```

# Usage

- To run the script:
```
python script.py --content <content file> --styles <style file>
```
